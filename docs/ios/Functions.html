The following functions are available globally.

  • fetch(_:method:parameters:headers:body:urlSessionDelegate:fetchQueue:completionQueue:completion:)

    The most generic of the three fetch calls, this one provides a completion handler that receives a value of Data or Error. The data can be converted in numerous ways but it is up to the calling user to do so.

    Declaration

    public func fetch(
  _ url: String,
  method: HTTPMethod = .get,
  parameters: [String:String]? = nil,
  headers: [HTTPHeader:String]? = nil,
  body: Data? = nil,
  urlSessionDelegate: URLSessionDelegate? = nil,
  fetchQueue: DispatchQueue = DispatchQueue.global(qos: .userInitiated),
  completionQueue: DispatchQueue? = nil,
  completion: @escaping (Data?, URLResponse?, Error?) -> Void
)

    Parameters

    url

    the full URL as a string that you want to request data from
    method

    one of the various HTTPMethod enum values
    parameters

    a [String:String] of url parameters to include
    headers

    a [HTTPHeader:String] dictionary of header keys and values
    body

    a Data object of content to send along as the HTTP body when posting data
    urlSessionDelegate

    a session delegate, often used for handling SSL issues and other edge case scenarios
    completion

    the handler of supplied user code that handles the Data, the URLResponse and/or the Error instances returned.
  • fetchJSON(_:method:parameters:headers:body:urlSessionDelegate:fetchQueue:completionQueue:completion:)

    The JSON variant of the three fetch calls, this one provides a completion handler that receives a value of [String:Any] or Error. The data can be converted in numerous ways but it is up to the calling user to do so.

    Attention

    This function does not handle JSON fragments. So if the top level value returned is not an Array or an Object, you may want to parse it manually using either fetchBody() or fetch()

  • fetchGraphql(_:urlParameters:gqlQuery:gqlVariables:urlSessionDelegate:fetchQueue:completionQueue:completion:)

    A utility function for querying GraphQL endpoints; provides a completion handler that receives a value of JSON or Error. Yields a FetchError if response cannot be parsed as JSON or contains a GraphQL error. Uses SwiftyJSON to JSONify the query and response.

  gqlQuery: String,

  fetchQueue: DispatchQueue = DispatchQueue.global(qos: .userInitiated),
  completionQueue: DispatchQueue? = nil,
  completion: @escaping (JSON?, URLResponse?, Error?) -> Void
)
