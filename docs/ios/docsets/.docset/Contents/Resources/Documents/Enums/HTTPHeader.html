HTTPHeader

public enum HTTPHeader : String

A concrete enum for all the listed HTTP header types provided by the Mozilla Developer Network headers page as of April 18th, 2019. In the generated documentation there is a tinyurl link that should evaluate to https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers. If, for whatever reason the tinyurl expires or goes bad, please update the docs with the above value.

  • credentials(_:_:_:_:)

    When .Authorization is used, it often requires the use of a String of a Base64 encoded string value. This function returns a string with the value of "<type> <value>" where type is often Basic or Bearer

    Declaration

    Swift

    public static func credentials(
  _ value: String,
  _ type: String = "Basic",
  _ encoding: HTTPHeaderBase64Encoding = .passthru,
  _ dict: HTTPHeaders? = nil
) -> HTTPHeaders

    Parameters

    value

    the value of the auth string, optionally not base64 encoded
    type

    the type of authorization, often Bearer or Basic, for example
    encoding

    an optional parameter that, by default, typically passes the value supplied through. If .encoded or .decoded are supplied, the proper encoding or decoding of the supplied value are performed before the final output is generated, but only for the value field
    dict

    an optional dictionary whose values are modified rather than creating a new one to hold said values.

    Return Value

    a string such as Basic … or Bearer … where … indicates the given, encoded or decoded string value
