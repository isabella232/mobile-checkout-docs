EventType

public enum EventType : String, CaseIterable

As new event types are identified, a new enum case should be defined here so that it can be listened to and fired.

  • gqlPayload

    The default event type in the application

    Declaration

    Swift

    case gqlPayload = "GraphQL Query Payload"
  • vgvLayout

    This event is fired when listening instances of VGV should be notified to redraw their contents.

    Declaration

    Swift

    case vgvLayout = "VerticalGrowViewLayout"
  • vgvSetViews

    This event takes a tuple of (ViewBuffers, [UIView]) as its data element and should cast the data of the event accordingly. The listener should consume the data and replace the contents of the view with the contents supplied

    Declaration

    Swift

    case vgvSetViews = "VerticalGrowViewSetBufferViews"
  • vgvAddViews

    This event takes a tuple of (ViewBuffers, [UIView]) as its data element and should cast the data of the event accordingly. The listener should consume the data and will append the contents of the view with the contents supplied

    Declaration

    Swift

    case vgvAddViews = "VerticalGrowViewAddBufferViews"
  • vgvUseView

    This event takes a ViewBuffers enum value and sets the current display in the vertical grow view with the contents specified.
