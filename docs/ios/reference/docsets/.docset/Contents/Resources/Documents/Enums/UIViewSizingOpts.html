UIViewSizingOpts

public enum UIViewSizingOpts : Equatable

The UIViewSizingOpts enumeration defines some of the ways that the following calculate*Heights functions work.

  • layoutCompressedSize

    Calls to systemLayoutSizeFitting(:) optionally takes a constant size if a known frame is not given. These can be either one of the UIView constants layoutFittingCompressedSize or layoutFittingExpandedSize; this option denotes the former

    Declaration

    Swift

    case layoutCompressedSize
  • preferSetSize

    There is likely a frame.size.height value set on the view itself, if present, we are stating that we prefer that value to others

    Declaration

    Swift

    case preferSetSize
  • preferSubviewSum

    In some cases it might make sense to recursively call calculateHeight on each of the subviews and return that value. preferSubView will be removed on recursive calls meaning that we only calculate one level down. Finally it should be noted that preferSubviewSum will be removed from the supplied flags if there are no subviews in the view being sized.

    Declaration

    Swift

    case preferSubviewSum
  • minimumOfChoices

    By default, when there is a preferred type of height, that value and the calculated value will be compared and the maximum of the two will be returned. If this option is present, the minimum of the two will be given instead.

    Declaration

    Swift

    case minimumOfChoices
  • includeYOffset

    Only applying to the calculation of subviews, if present any subviews y coordinate is also added to the final height, allowing for any calculated whitespace.

    Declaration

    Swift

    case includeYOffset
  • temporarilyDisableConstraints

    AutoLayout constraints can and often do affect the size value returned by the methods used to query for height. If this option is present, any active constraints are temporarily disabled while the sizes are being recalculated. Once the calculation is complete, they are reenabled.

    Avoid setting dirty flags for code that mutates the state of the object
