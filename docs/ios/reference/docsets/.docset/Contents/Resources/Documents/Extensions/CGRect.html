  • xy

    This property when gotten is a two value tuple with the frame’s origin’s x and y values. When set, it receives a two value tuple with the same values and uses them to create a new CGPoint. The new point and existing size are used to make a new CGRect value for the view’s frame.

    Declaration

    Swift
