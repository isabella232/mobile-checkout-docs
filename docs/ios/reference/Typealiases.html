Type Aliases

The following type aliases are available globally.

  • HTTPHeaders

    This simple type alias makes it easy to handle the instantiation of the collection that is core to passing around headers and their values

    Declaration

    Swift

    public typealias HTTPHeaders = [HTTPHeader : String]
  • NSASMap

    The types are a bit long and as a shortcut, this makes much of the code far more readable.

    Declaration

    Swift

    public typealias NSASMap = [NSAttributedString.Key : Any]

  • GestureClosureWrapperFn

    Undocumented

    Declaration

  • GraphQLResultHandler

    A handler for operation results.

    Declaration

    Swift

    public typealias GraphQLResultHandler<Operation> = (_ result: GraphQLResult<Operation.Data>?, _ error: Error?, _ service: GraphQLService?) -> () where Operation : GraphQLOperation

    Parameters

    result

    The result of the performed operation, or nil if an error occurred.
    error

    An error that indicates why the mutation failed, or nil if the mutation was succesful.
    service

    the GraphQLService in question
  • GQLDataClosure

    A shorthand for the closure that is invoked whenever data is updated from the graphql call for the GetuserAndCheckoutQuery() query.

    Declaration

    Swift

    public typealias GQLDataClosure = (PayloadQuery.Data?, GQLUserData?, Error?) -> Void
  • GQLDataEvent

    For every event that gets fired, we will store these (think about storing the last 10 or something in the future) and every new listener added will immediately receive all the previously played events that contain no errors

    Declaration

    Swift

    public typealias GQLDataEvent = [(PayloadQuery.Data?, GQLUserData?, Error?)]

  • ECTokenCall

    The callback for ECToken.generate(). Generates three types of ECTokens authorize, sale and order. The URLResponse and/or Error are included in the case that such values are available.

    Declaration

    Swift

    public typealias ECTokenCall = ((String?, String?, String?)?, URLResponse?, Error?) -> Void
  • EventID

    Used as a way to uniquely identify a listener. As long as this EventID is always the same when the listener is registered the event handler that is registered will be overwritten and it can be defined functions or places that are reentrant

    Declaration

    Swift

  • LocaleTuple

    Used throughout this class and other places in the SDK, the LocaleTuple is simply a string following the format of (identifier, language, region) e.g. (en_US, en, US)

    Declaration

    Swift

    public typealias LocaleTuple = (String, String, String)
