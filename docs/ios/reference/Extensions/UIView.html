  • animations(_:_:_:_:_:_:_:_:_:_:_:_:_:_:_:_:)

    The problem with UIView.animate and UIView.animateKeyFrames is that they do not allow for code to be executed between each sequenced animation. Due to the way that most of the basic animation libraries work, there is a contrived limit to the number of linked animations that can be performed.

    This extension allows up to four animations and completion blocks to be chained in order. The first has to complete before the second starts, and so on.

    If you wish to return .intrinsicContentSize if only if both values are not equal to UIView.noIntrinsicMetric, then specify true to the second parameter ensureBothAreIntrinsic

    Declaration

  _ size: CGSize,
  ensureBothAreIntrinsic: Bool = false
) -> CGSize

    size

    the default size to use in the case that the intrinsic content size has not been taken
    ensureBothAreIntrinsic

    true if both values have to not be equal to UIView.noIntrinsicMetric or false if only is intrinsic; the later is the default

    Return Value

    the intrinsicContentSize or supplied size if either or both the intrinsic height and width are not equal to UIView.noIntrinsicMetric
