  • setAltHeader(_:_:)

    This method will take all the subviews in the supplied view and add them to the header after the headers existing subviews are removed. They are removed by calling .removeFromSuperview() on each element. Likewise, the view supplied will not be used, but its contents will be moved in to the existing header view.

    Note

    Note:
