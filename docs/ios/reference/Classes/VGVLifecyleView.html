VGVLifecyleView

public class VGVLifecyleView : UIView, VGVLifecycle

The VGVLifecycleView or VerticalGrowLifecycleView is an easy to use container that invokes the various VGVLifecycle events and can be used wherever a basic UIView is used to wrap contents that are added to a VerticalGrowView

      Rather than extending this class for any particular callback, this convenience initializer allows the instances to be created with closures that suffice for the same purpose. By default all these values are nil. If the closures are nil, the overloaded methods are invoked instead.
